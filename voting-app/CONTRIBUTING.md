# Contributing

Contributions are welcome but not expected or required. If you have improvements you'd like to share, feel free to open a pull request. However, there is no guarantee it will be reviewed or merged.

Forks are encouraged — take the code, make it your own, and use it however you like.

## Pull requests

This is a volunteer project with no maintenance commitment. PRs may be reviewed if and when the author has time. If your PR is not reviewed, please feel free to maintain your own fork.

## Code style

- Python: clean, readable, well-commented
- Templates: plain HTML, no JavaScript in the voter flow
- CSS: mobile-first, conservative (no CSS features newer than 2018)
- Tests: pytest, test every rule and threshold
